# Memory Store

The Memory Store provides conversation persistence using Cloudflare D1.

## Usage

```ts
import { createMemoryStore, toAgentMessages } from 'hareai'

// Create a memory store with your D1 binding
const memory = createMemoryStore({ db: env.DB })

// Save a message
await memory.saveMessage({
  conversationId: 'conv_123',
  role: 'user',
  content: 'Hello!',
})

// Get conversation history
const messages = await memory.getMessages({
  conversationId: 'conv_123',
  limit: 50,
})

// Convert to agent-compatible format
const agentMessages = toAgentMessages(messages)
```

## Integration with Agents

### With HareEdgeAgent

```ts
import { createMemoryStore, toAgentMessages, HareEdgeAgent } from 'hareai'

const memory = createMemoryStore({ db: env.DB })

// Load conversation history
const messages = await memory.getMessages({
  conversationId: 'conv_123',
  limit: 50,
})

const agent = new HareEdgeAgent({
  name: 'My Agent',
  instructions: 'You are a helpful assistant.',
  model: '@cf/meta/llama-3.3-70b-instruct-fp8-fast',
  ai: env.AI,
})

// Include history in the conversation
const response = await agent.stream([
  ...toAgentMessages(messages),
  { role: 'user', content: 'Continue our conversation' }
])

// Save the response
await memory.saveMessage({
  conversationId: 'conv_123',
  role: 'assistant',
  content: await response.text,
})
```
