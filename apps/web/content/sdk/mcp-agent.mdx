# HareMcpAgent

`HareMcpAgent` implements the Model Context Protocol (MCP), allowing external AI clients like Claude Desktop or Cursor to use Hare's tools.

**Workers-only**: This class requires the Cloudflare Workers runtime. Import from `hareai/workers`.

## What is MCP?

[Model Context Protocol](https://modelcontextprotocol.io) is an open standard for connecting AI assistants to external tools and data sources. With `HareMcpAgent`, you can:

- Connect Claude Desktop to your Hare workspace
- Use Hare tools from Cursor IDE
- Integrate with any MCP-compatible client

## Usage

```ts
import { HareMcpAgent } from 'hareai/workers'

// Export as Durable Object
export { HareMcpAgent }

export default {
  async fetch(request: Request, env: Env) {
    const id = env.MCP_AGENT.idFromName('workspace-123')
    const agent = env.MCP_AGENT.get(id)
    return agent.fetch(request)
  }
}
```

## Connecting Clients

### Claude Desktop

Add to your Claude Desktop config (`claude_desktop_config.json`):

```json
{
  "mcpServers": {
    "hare": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://your-app.workers.dev/api/mcp/workspace-123"]
    }
  }
}
```

### Cursor IDE

Add to Cursor settings:

```json
{
  "mcp.servers": {
    "hare": {
      "url": "https://your-app.workers.dev/api/mcp/workspace-123"
    }
  }
}
```

## Extending HareMcpAgent

Add custom tools by extending the class:

```ts
import { HareMcpAgent } from 'hareai/workers'
import type { McpAgentEnv } from 'hareai/workers'

class MyMcpAgent extends HareMcpAgent {
  async init(): Promise<void> {
    await super.init()

    // Add custom tool
    this.server.tool(
      'my_custom_tool',
      'Does something custom',
      { type: 'object', properties: { input: { type: 'string' } } },
      async (params) => {
        return {
          content: [{ type: 'text', text: `Processed: ${params.input}` }]
        }
      }
    )
  }
}

export { MyMcpAgent as HareMcpAgent }
```
