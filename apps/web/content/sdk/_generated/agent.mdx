## HareAgent

HareAgent - A Cloudflare Agents SDK implementation.

This class extends the Agent base class to provide:
- Stateful conversations with persistence
- Real-time WebSocket communication
- Tool execution
- Scheduled tasks

  }

## HareMcpAgent

HareMcpAgent - Exposes Hare tools via Model Context Protocol.

MCP allows external AI clients (Claude, Cursor, etc.) to use
Hare's tools in a standardized way.
