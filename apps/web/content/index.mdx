# Hare Documentation

Welcome to the Hare documentation. Hare is an AI agent platform that runs on Cloudflare's edge network, providing low-latency AI capabilities globally.

## What is Hare?

Hare enables you to:

- **Build AI Agents** - Create intelligent agents with custom tools and capabilities
- **Deploy to the Edge** - Run your agents on Cloudflare's global network
- **Scale Automatically** - Handle any load with Durable Objects and Workers AI

## Getting Started

- [Quick Start](/docs/guides/getting-started) - Get up and running in 5 minutes
- [SDK Reference](/docs/sdk) - Explore the @hare/agent SDK

## Architecture

Hare is built on:

- **Cloudflare Workers** - Serverless compute at the edge
- **Durable Objects** - Stateful coordination for agents
- **Workers AI** - Inference with Llama, Mistral, and more
- **Vercel AI SDK** - Streaming and tool calling support
