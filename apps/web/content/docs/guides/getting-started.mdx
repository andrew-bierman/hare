# Getting Started

This guide will walk you through creating your first AI agent with Hare.

## Prerequisites

- [Bun](https://bun.sh) or Node.js 20+
- A Cloudflare account with Workers AI enabled
- Basic TypeScript knowledge

## Create a New Project

```bash
# Create a new Cloudflare Workers project
bun create cloudflare my-agent

cd my-agent

# Install the Hare SDK
bun add @hare/agent
```

## Configure Bindings

Update your `wrangler.toml`:

```toml
name = "my-agent"
main = "src/index.ts"
compatibility_date = "2024-12-01"

[ai]
binding = "AI"
```

## Create Your Agent

Replace `src/index.ts`:

```ts
import { createEdgeAgent, createWorkersAIModel } from '@hare/agent'

interface Env {
  AI: Ai
}

export default {
  async fetch(request: Request, env: Env): Promise<Response> {
    const agent = createEdgeAgent({
      model: createWorkersAIModel({
        modelId: '@cf/meta/llama-3.3-70b-instruct-fp8-fast',
        ai: env.AI,
      }),
      systemPrompt: 'You are a helpful AI assistant.',
    })

    // Parse the message from the request
    const { message } = await request.json<{ message: string }>()

    // Get the response
    const response = await agent.chat(message)

    return new Response(await response.text, {
      headers: { 'Content-Type': 'text/plain' },
    })
  },
}
```

## Run Locally

```bash
bun run dev
```

## Test Your Agent

```bash
curl http://localhost:8787 \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello! What can you help me with?"}'
```

## Deploy

```bash
bun run deploy
```
