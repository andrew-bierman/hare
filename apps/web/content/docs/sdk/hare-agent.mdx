# HareAgent

`HareAgent` is a full-featured AI agent built on Cloudflare's Agents SDK with Durable Objects for state persistence.

**Workers-only**: This class requires the Cloudflare Workers runtime.

## Features

- **Durable Object State** - Persistent state across requests
- **WebSocket Support** - Real-time bidirectional communication with hibernation
- **Tool Execution** - 40+ built-in tools
- **Scheduling** - One-time and recurring tasks via alarms
- **State Sync** - Automatic broadcast to connected clients

## Usage

```ts
import { HareAgent } from '@hare/agent/workers'

// Export as Durable Object
export { HareAgent }

export default {
  async fetch(request: Request, env: Env) {
    // Get or create agent instance
    const id = env.HARE_AGENT.idFromName('my-agent')
    const agent = env.HARE_AGENT.get(id)
    return agent.fetch(request)
  }
}
```

## HTTP Endpoints

HareAgent exposes these endpoints:

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/health` | GET | Health check |
| `/state` | GET | Get current state |
| `/configure` | POST | Update agent configuration |
| `/chat` | POST | Chat via HTTP (streaming SSE) |
| `/schedules` | GET | List scheduled tasks |

### Chat via HTTP

```ts
const response = await fetch('https://your-agent.workers.dev/chat', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    message: 'Hello!',
    userId: 'user_123',
  }),
})

// Response is Server-Sent Events
const reader = response.body.getReader()
const decoder = new TextDecoder()

while (true) {
  const { done, value } = await reader.read()
  if (done) break
  console.log(decoder.decode(value))
}
```

## WebSocket Protocol

Connect via WebSocket for real-time communication:

```ts
const ws = new WebSocket('wss://your-agent.workers.dev')

// Receive messages
ws.onmessage = (event) => {
  const message = JSON.parse(event.data)
  // message.type: 'text' | 'tool_call' | 'tool_result' | 'state_update' | 'error' | 'done'
  console.log(message)
}

// Send chat message
ws.send(JSON.stringify({
  type: 'chat',
  payload: {
    message: 'Hello!',
    userId: 'user_123',
  }
}))
```

### Client Message Types

| Type | Payload | Description |
|------|---------|-------------|
| `chat` | `ChatPayload` | Send a chat message |
| `configure` | `Partial<HareAgentState>` | Update configuration |
| `execute_tool` | `ToolExecutePayload` | Execute a tool |
| `get_state` | none | Request current state |
| `schedule` | `SchedulePayload` | Schedule a task |

### Server Message Types

| Type | Data | Description |
|------|------|-------------|
| `text` | `{ content: string }` | Streaming text chunk |
| `tool_call` | `{ toolId, params }` | Tool execution started |
| `tool_result` | `{ toolId, result }` | Tool execution completed |
| `state_update` | `HareAgentState` | State changed |
| `error` | `{ message: string }` | Error occurred |
| `done` | `{ messageCount }` | Response complete |

## Scheduling

Schedule one-time or recurring tasks:

```ts
// One-time task (5 minutes from now)
ws.send(JSON.stringify({
  type: 'schedule',
  payload: {
    action: 'sendReminder',
    executeAt: Date.now() + 5 * 60 * 1000,
    payload: { message: 'Time for a break!' }
  }
}))

// Recurring task (every hour)
ws.send(JSON.stringify({
  type: 'schedule',
  payload: {
    action: 'runMaintenance',
    cron: '0 * * * *',
  }
}))
```

## State

```ts
interface HareAgentState {
  agentId: string
  workspaceId: string
  name: string
  instructions: string      // System prompt
  model: string             // e.g., '@cf/meta/llama-3.3-70b-instruct-fp8-fast'

  isProcessing: boolean
  lastActivity: number
  connectedUsers: string[]
  scheduledTasks: ScheduledTask[]
  status: 'idle' | 'processing' | 'error'
  lastError?: string
}
```

## Extending HareAgent

Create a custom agent by extending the class:

```ts
import { HareAgent, type HareAgentEnv } from '@hare/agent/workers'

interface MyEnv extends HareAgentEnv {
  MY_KV: KVNamespace
}

class MyCustomAgent extends HareAgent<MyEnv> {
  async onStart(): Promise<void> {
    await super.onStart()
    // Add custom initialization
  }

  async executeScheduledTask(data: { action: string }) {
    if (data.action === 'myCustomAction') {
      // Handle custom scheduled action
      return
    }
    await super.executeScheduledTask(data)
  }
}

export { MyCustomAgent as HareAgent }
```
