# Tools

The `hareai` SDK includes 59+ built-in tools that agents can use to interact with Cloudflare services and perform various operations.

## Getting Tools

```ts
import { getSystemTools, getToolsByCategory } from 'hareai/tools'

// Get all system tools
const allTools = getSystemTools({
  env,
  workspaceId: 'my-workspace',
  userId: 'user-123'
})

// Get tools by category
const kvTools = getToolsByCategory({
  category: 'kv',
  context: { env, workspaceId: 'my-workspace', userId: 'user-123' }
})
```

## Creating Custom Tools

```ts
import { createTool, success, failure } from 'hareai/tools'
import { z } from 'zod'

const weatherTool = createTool({
  id: 'get_weather',
  description: 'Get current weather for a location',
  inputSchema: z.object({
    city: z.string().describe('City name'),
    units: z.enum(['celsius', 'fahrenheit']).default('celsius')
  }),
  execute: async (params, context) => {
    try {
      const weather = await fetchWeather(params.city, params.units)
      return success({ temperature: weather.temp, conditions: weather.conditions })
    } catch (error) {
      return failure(`Failed to fetch weather: ${error.message}`)
    }
  }
})
```

## Using Tools with Agents

### With HareEdgeAgent

```ts
import { HareEdgeAgent } from 'hareai'
import { getSystemTools } from 'hareai/tools'

const tools = getSystemTools({
  env,
  workspaceId: 'my-workspace',
  userId: 'user-123'
})

const agent = new HareEdgeAgent({
  name: 'My Agent',
  instructions: 'You are a helpful assistant with access to many tools.',
  model: '@cf/meta/llama-3.3-70b-instruct-fp8-fast',
  ai: env.AI,
  tools,
})
```

### With HareAgent (Durable Object)

HareAgent automatically loads system tools. To execute a tool via WebSocket:

```ts
ws.send(JSON.stringify({
  type: 'execute_tool',
  payload: {
    toolId: 'kv_get',
    params: { key: 'my-key' }
  }
}))
```
